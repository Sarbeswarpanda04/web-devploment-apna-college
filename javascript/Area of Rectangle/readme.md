# ▭ Rectangle Area Visualizer

An interactive web application built with **HTML**, **CSS**, **JavaScript**, and **SVG** that calculates the area of a rectangle and visualizes it dynamically.

## 🚀 Features

- 📏 Enter the **Length** and **Width** of a rectangle.
- 🧮 Calculate the area instantly.
- 🖼️ Draw the rectangle using **SVG**.
- 📐 Display the length and width labels.
- 📊 Show the calculated area inside the rectangle.
- 📱 Responsive and modern user interface.
- 🔄 Automatically scales the rectangle to fit the drawing area while preserving its proportions.

## 📐 Formula


::contentReference[oaicite:0]{index=0}


For a rectangle:

**Area = Length × Width**

Where:
- **Length (L)** = Horizontal side
- **Width (W)** = Vertical side
